3. Comandos básicos de Git aprendidos:
git status - Ver el estado de los archivos
git add . - Agregar todos los cambios
git add archivo.py - Agregar un archivo específico
git commit -m "mensaje" - Guardar cambios con descripción
git push - Subir cambios a GitHub
git pull - Obtener últimos cambios
git log - Ver historial de commits
git checkout -- archivo - Descartar cambios no guardados

4. Flujo de trabajo establecido:
1) Antes de trabajar: git pull
2) Después de modificar:
   - git status
   - git add .
   - git commit -m "descripción"
   - git push
